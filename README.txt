Casse-brique dans lequel de nouveaux rangs de briques apparaissent toutes les 10 secondes.
Le but est de tenir le plus longtemps possible et d'atteindre le plus haut score.

Controles :
Flèches gauche / droite + espace

Bonus :
M : machine gun (espace pour tirer)
Flèche vers le haut : fait remonter les briques d'un rang
+ : augmente la taille de la barre
- : diminue la taille de la barre

Développé sous Godot 3.0.2
